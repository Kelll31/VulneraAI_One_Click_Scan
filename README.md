# VulneraAI One Click Scan

Одностраничный сайт для быстрого заказа сканирования безопасности сайтов.

## Описание

Это упрощенная версия VulneraAI для быстрого заказа сканирования безопасности. Пользователь просто вводит URL сайта, свой email и выбирает тип сканирования, после чего переходит к оплате.

## Структура проекта

```
.
├── index.html          # Главная страница с формой заказа
├── payment.html        # Страница оплаты
├── success.html        # Страница успешной оплаты
├── css/
│   ├── style.css       # Основные стили (из основного проекта)
│   └── payment.css     # Стили для страниц оплаты
├── js/
│   ├── scan.js         # Логика формы заказа
│   ├── payment.js      # Логика оплаты
│   └── success.js      # Логика страницы успеха
└── README.md
```

## Особенности

- Минималистичный интерфейс с фокусом на простоту использования
- Использует те же стили, что и основной сайт VulneraAI
- Три типа сканирования:
  - Быстрое (500 ₽) - базовые проверки, 5-10 минут
  - Полное (1500 ₽) - углубленный анализ, 30-60 минут
  - Комплексное (3000 ₽) - максимальный охват, 2-4 часа
- Интеграция с API VulneraAI (`http://91.201.252.14/api`)
- Адаптивный дизайн для всех устройств

## API

### Создание быстрого сканирования

```
POST /api/quick-scan
```

**Тело запроса:**
```json
{
  "website": "https://example.com",
  "email": "user@example.com",
  "scan_type": "quick",
  "payment_method": "card",
  "amount": 500
}
```

**Ответ:**
```json
{
  "success": true,
  "scan_id": "abc123",
  "message": "Сканирование запущено"
}
```

## Установка

1. Клонируйте репозиторий
2. Откройте `index.html` в браузере или разверните на веб-сервере

## Разработка

Для локальной разработки можно использовать любой простой HTTP-сервер:

```bash
# Python 3
python -m http.server 8000

# Node.js (npx)
npx http-server

# PHP
php -S localhost:8000
```

## Интеграция с платежными системами

В текущей версии используется заглушка для оплаты. Для продакшена необходимо интегрировать:
- ЮKassa / CloudPayments для банковских карт
- ЮMoney API
- QIWI API

# VulneraAI One Click Scan

Одностраничный сайт для быстрого заказа пентеста веб-сайтов.

## Описание

Это упрощенная версия VulneraAI для быстрого заказа полного пентеста веб-сайта. Пользователь просто вводит URL сайта и свой email, после чего переходит к оплате.

## Структура проекта

```
.
├── index.html          # Главная страница с формой заказа
├── payment.html        # Страница оплаты
├── success.html        # Страница успешной оплаты
├── css/
│   ├── style.css       # Основные стили
│   └── payment.css     # Стили для страниц оплаты
├── js/
│   ├── scan.js         # Логика формы заказа
│   ├── payment.js      # Логика оплаты
│   └── success.js      # Логика страницы успеха
└── README.md
```

## Особенности

- Минималистичный интерфейс с фокусом на простоту использования
- Использует те же стили, что и основной сайт VulneraAI
- Один тип сканирования - полный пентест (1500 ₽)
- Центрированная форма без скроллинга
- Интеграция с API VulneraAI (`http://91.201.252.14/api`)
- Адаптивный дизайн для всех устройств

## Услуга

**Полный пентест веб-сайта - 1500 ₽**

- ✓ Поиск уязвимостей OWASP Top 10
- ✓ Анализ конфигурации сервера
- ✓ Проверка SSL/TLS
- ✓ Детальный отчет с рекомендациями
- Время выполнения: 30-60 минут

## API

### Создание быстрого сканирования

```
POST /api/quick-scan
```

**Тело запроса:**
```json
{
  "website": "https://example.com",
  "email": "user@example.com",
  "scan_type": "full",
  "payment_method": "card",
  "amount": 1500
}
```

**Ответ:**
```json
{
  "success": true,
  "scan_id": "abc123",
  "message": "Сканирование запущено"
}
```

## Установка

1. Клонируйте репозиторий
2. Откройте `index.html` в браузере или разверните на веб-сервере

## Разработка

Для локальной разработки можно использовать любой простой HTTP-сервер:

```bash
# Python 3
python -m http.server 8000

# Node.js (npx)
npx http-server

# PHP
php -S localhost:8000
```

## Интеграция с платежными системами

В текущей версии используется заглушка для оплаты. Для продакшена необходимо интегрировать:
- ЮKassa / CloudPayments для банковских карт
- ЮMoney API
- QIWI API
